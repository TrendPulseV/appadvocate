# AppAdvocate

> Reviews that mean something.

A vetted advocacy network. We help developers earn 5 stars — or get the exact,
actionable feedback that tells them exactly why they didn't.

## The idea

A user found a critical bug in a founder's app. He emailed — he didn't have to.
The founder gave him a year of Pro free. Then he thought: what if he could find
people like him *before* the bug ever ships?

That's AppAdvocate.

## Domains
- appadvocate.io (primary)
- appadvocate.app
- appadvocate.dev
- earnhq.app (alt brand)

## Stack
- Next.js 14 App Router
- Tailwind CSS
- Vercel deployment
- Beehiiv (waitlist CRM)
